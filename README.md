# Ayub's Homelab Project
Personal homelab built on Proxmox VE running Debian, Docker, Portainer, Immich, and Tailscale for remote management and self-hosted cloud services.

# Homelab

This repository documents my personal homelab running on an HP Z440 workstation.

## Technologies Used
- Proxmox VE
- Debian Linux
- Docker
- Portainer
- Tailscale
- Immich

## Features
- Virtual machine management with Proxmox
- Secure remote access using Tailscale
- Self-hosted photo backup and management with Immich
- Docker container deployment and management
- Family photo storage and synchronization
- Snapshot and backup planning

## What I Learned
- Linux system administration
- Virtualization and VM management
- Docker containers and networking
- Remote access and VPN technologies
- Troubleshooting networking and storage issues
- Self-hosting and infrastructure management

## Hardware
- HP Z440 Workstation
- 1TB HDD
- 500GB Samsung SSD
